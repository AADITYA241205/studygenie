# Changelog

All notable changes to StudyGenie will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial release of StudyGenie
- PDF and image document upload with OCR support
- AI-powered summary generation using Google Gemini
- Dynamic quiz generation with 3 difficulty levels (Easy, Medium, Hard)
- Interactive flashcard system with spaced repetition
- AI tutor chat for document-specific questions
- Multi-language support (English, Hindi, Marathi, Spanish, French, German)
- User authentication and progress tracking
- Learning streak and score analytics
- Admin panel for content management
- Demo user accounts for testing

### Features
- Document processing pipeline with text extraction
- Context-aware AI content generation
- Responsive Bootstrap 5 UI
- SQLite database for easy deployment
- Batch file automation for Windows setup

### Tech Stack
- Django 4.2
- Google Gemini 1.5 Flash
- PyPDF2 for PDF processing
- Pytesseract for OCR
- Bootstrap 5 for frontend
